\# searchfun
# 美食收集
## 功能
### 前台功能：
* 美食餐廳搜尋、介紹
* 地圖圖資切換
* 地標顯示
* 定位、距離計算
* 結合 google 導航
* 收藏功能
### 後台功能：
* 地標管理(CRUD)
* 管理者註冊、登入
## 使用技術：
### 前端：
* Vue CLI 4 建立與打包專案
* Vue.js 規劃 SPA 架構
* 串接 API 資料存取
* Vue Router 建構路由
* Bootstrap 4 + SCSS
* leaflet 套件運用
* Mapbox 地圖使用
### 後端：
* nodejs express 使用
* 建立 RESTful API
* 串接 MongoDB 資料庫
* 使用 bcryptjs 做密碼加解密處理
* passport.js 套件登入驗證
* ejs 樣板使用
* heroku 部署使用
* 因 heroku mlab 功能取消，因此改用 firebase 資料庫處理
